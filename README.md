# Cadastro_Alunos

Projeto desenvolvido em Angular com integração ao MySQL. Possui uma página de cadastro de alunos, permitindo inserir, editar, listar e excluir registros diretamente no banco de dados.

## Servidor de desenvolvimento

Para iniciar o servidor de desenvolvimento, rode:

`ash
ng serve
`

Abra http://localhost:4200/ no navegador. A aplicação recarrega automaticamente quando arquivos de origem são modificados.

## Build

Para gerar uma versão de produção:

`ash
ng build
`

Os artefatos de build são gerados na pasta dist/.

## Testes

Execute testes unitários com:

`ash
ng test
`
